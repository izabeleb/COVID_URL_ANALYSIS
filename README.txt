This directory contains all project related data and program files. All raw collected data has been removed to keep with Twitter data privacy. All API keys and logins have been removed, some script functionality may be limited.

"Experiment_Data" contains the resulting evidence of all data collection logging, model tuning, and script processing. Most of this data is presented in chart format in the final report.

"Final_Deliverable" contains the tuned model and prediction script written to accept data csv inputs, as well as all peripheral files required for prediction.

"Processing_and_Collection_Scripts" contains the scripts used for data collection, processing, and model creation, as well as unit tests for parts of the system.
